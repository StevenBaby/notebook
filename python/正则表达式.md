# 正则表达式

| 模式 | 描述 |
| - | - |
| ^ | 匹配字符串的开头 |
| $ | 匹配字符串的末尾 |
| (...) | 分组 |
| (?:...) | 不分组 |
| (?#...) | 注释 |
| (?=...) | 之后的字符串内容需要 **匹配** 表达式才能成功匹配，不消耗字符串内容 |
| (?!...) | 之后的字符串内容需要 **不匹配** 表达式才能成功匹配，不消耗字符串内容 |
| (?<=...) | 之前的字符串内容需要 **不匹配** 表达式才能成功匹配，不消耗字符串内容 |
| (?<!...) | 之前的字符串内容需要 **不匹配** 表达式才能成功匹配，不消耗字符串内容 |
